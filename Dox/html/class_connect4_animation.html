Detailed Description

Class which handles the animations for connect four.

\file - Connect4Animation.java

Author
Daniel Squires - 709547
Date
22/03/2014

This works by painting a "glasspane" over the application and painting to that every time cycle is called, which happens when the timer ticks over.

The animation process is started when a move is made in Connect4 and animate is called, which then sets everything up and starts the timer
