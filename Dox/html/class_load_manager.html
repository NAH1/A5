Detailed Description

LoadManger deals with loading a saved game from a file.

\file LoadManager.java

Author
Gavin Tsang 658679
Date
27/03/2014

LoadManager takes in a text File named "SaveGame.txt" which has been generated by the SaveManager class upon the saving of a game. The file is then parsed and the data passed into the game to reconstruct the game at the state it was most recently saved. Any errors within the save file will cause an error message to display.
