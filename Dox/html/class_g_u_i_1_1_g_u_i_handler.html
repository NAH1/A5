Detailed Description

This class is used by the main JFrame to provide methods for the buttons to call.

Author
Gavin Tsang

This class contains methods used by the JFrame. Any user interaction between the GUI get's sent to this class which then processes the action and passes the input to the correct subsystem within the game.
